# Dataset: Arvicole – co-catture in trappole

**Link al dataset originale:**  
https://networkrepository.com/mammalia-voles-bhp-trapping-27.php

Il dataset è stato manipolato per sfoltirlo e per invertire il peso.

## Descrizione
-Rete basata su co-catture di arvicole, ongi nodo è un arvicola
il peso è il  numero di co-catture invertito per rappresentare la distanza sociale tra le arvicole.  

-Dato che si formano gruppi di arvicole che non sono mai stati co-catturati, il risultato è una foresta.
Se impostiamo un valore di distanza sociale massimo **d**, si formerebbero dei cluster che rappresentano i gruppi sociali
distanziati al massimo di **d**.
